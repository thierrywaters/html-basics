How to make a simple header.

by Thierry Waters

In this tutorial, I am going to be showing you how to make a simple sticky header.

We're going to start by adding a div with a class of header.

<div class="header"></div>

Next we are going to add some text inside the header. I'm going to use a <h3> tag. After that we can add some nav links. To do this we must create an unorganised list <ul> and inside that we are going to add some list items <li>.

Your code should now look something like this.

<div class="header">
    <h3>My Header</h3>
    <ul>
        <li></li>
        <li></li>
        <li></li>
    </ul>
 </div>

This will create a list of all of our nav links. Now we will add some anchor tags <a> inside of the list items <li> with a "#" as the href. 

<div class="header">
    <h3>My Header</h3>
    <ul>
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
        <li><a href="#">Contact</a></li>
    </ul>
 </div>

It should look like this now.

Now that we've got the outline done in HTML it's time to add some styles using CSS.

First set all of the margins and padding to 0

* {
    margin: 0;
    padding: 0;    
}

In your CSS you will want to add a height of 50px to the class "header". Then choose a background color, I chose gray, and a foreground color, I chose white. Your CSS should look like this.

* {
    margin: 0;
    padding: 0;
}
.header {
    height: 50px;
    background: gray;
    color: white;
}

Then in the class header add display: flex;, this will make the items in "header" go across instead of down. Then add justify-content: space-between;, this will make the two objects space out to the opposite sides of the screen. Its a little bit to close so we can add some padding to fix that. Add and padding of "0" on top and bottom and "50px" on each side. Your code should look like this now.

* {
    margin: 0;
    padding: 0;
}
                
.header {
    height: 50px;
    background: gray;
    color: white;
    display: flex;
    justify-content: space-between;
    padding: 0 50px;
}

Now we are going to style the anchor tags. We are going to remove the text text decoration by using text-decoration: none;, then change the colour to white by using color: white;, and finally we are goin to add some padding by using padding: 10px 20px;. Your CSS should look like this now.

* {
    margin: 0;
    padding: 0;
}
                
.header {
    height: 50px;
    background: gray;
    color: white;
    display: flex;
    justify-content: space-between;
    padding: 0 50px;
}

.header a {
    text-decoration: none;
    color: white;
    padding: 10px 20px;
}

Now lets add a hover animation. We do it like this.

.header a:hover {
}

Now we can add the parameters we want to change. I'm going to change the background colour.

.header a:hover {
    background: black;
}

Now we can add the transition duration im going to add this, transition: background 0.4s ease;. This will change the background color with a duration of 0.4 seconds smoothly. Your CSS should look like this.

* {
    margin: 0;
    padding: 0;
}
    
.header {
    height: 50px;
    background: gray;
    color: white;
    display: flex;
    justify-content: space-between;
    padding: 0 50px;
}

.header a {
    text-decoration: none;
    color: white;
    padding: 10px 20px;
    transition: background 0.4s ease;
}

.header a:hover {
    background: black;
}

Now we are going to style the unorganised list. Type this .header ul {}, and add list-style: none; to remove the dots, display: flex; to make the li's group in a row, a width of 250px and finally type justify-content: space-between; which will space the li's out evenly.

* {
    margin: 0;
    padding: 0;
}
.header {
    height: 50px;
    display: flex;
    justify-content: space-between;
    align-items: center;
    padding: 0 50px;
    background: gray;
    color: white;
}
    
.header a {
    text-decoration: none;
    color: white;
    padding: 10px 20px;
    transition: background 0.4s ease;
}
    
.header a:hover {
    background: black;
}
    
.header ul {
    list-style: none;
    display: flex;
    width: 250px;
    justify-content: space-between;
}

Well done! You have now created your very own header using HTML and CSS.
