How to position elements.

by Thierry Waters

In this tutorial, I am going to be showing you how to postion HTML element on the screen. There are three types of positioning.

  • Relative Positioning
  • Absolute Positioning
  • Fixed Positioning

Relative Positioning

Relative positioning changes the position of the HTML element relative to where the element normally is. left: 20px; adds 20 pixels to the left position of the element. If you use negative values you can move the element to the right. eg: left: -20px;. You can only change the left and top values.

Absolute Positioning

An element with absolute positioning is positioned relative to your screen top-left corner. left: 20px; moves the element 20 pixels from the right of the screen. You can only change the left and top values.

Fixed Positioning

Fixed positioning lets you fix the position of an element to a particular spot on the page, even if you scroll. The left and top values will be relative to the screen. left: 20px; moves the element 20 pixels from the right of the screen. You can only change the left and top values.
